# np2stl
Create 3D STL files from numpy data

## Installation:
1. Clone the repository
```bash
git clone git@github.com:lhalve/np2stl.git
```
2. Add this repository to your PYTHONPATH environment variable
```bash
cd np2stl
export PYTHONPATH="${PYTHONPATH}:$PWD"
```
